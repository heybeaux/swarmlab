# Experiment 07 — Team's Choice

**Faculty under test:** you decide.

## The Brief

This one is yours. Beaux explicitly asked the team to invent an experiment of its own,
in addition to the selected set. Design something weird that stress-tests a heybeaux
faculty (Parliament, Lattice, Sonder, Engram, ACR/AWM) in a way experiments 03/04/06 do not.

## Requirements

- Write a real spec (`specs/07-<your-name>.md` may replace this file's title) before building.
- Justify it: which faculty, what hypothesis, why it teaches something the others don't.
- Build it on `core/` like the rest — traceable, replayable, scored.
- Be genuinely creative. Candidate seeds from the original brainstorm you could riff on or
  ignore: telephone-game compiler, gossip/rumor mill, economic agents under token scarcity,
  reverse-engineer-the-black-box, schema negotiation. Or invent something entirely new.

## Definition of Done

Spec written, experiment built on `core/`, runs, valid trace, README with hypothesis +
what you observed + which faculty it informs, committed, journal note.
